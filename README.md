# Brew & Co. — Infrastructure

Terraform configuration for Brew & Co. cloud resources.

## Structure

| File | Description |
|------|-------------|
| `main.tf` | Main resource definitions |
| `variables.tf` | Input variables |
| `outputs.tf` | Output values |
| `environments/dev.tfvars` | Dev environment values |
| `environments/prod.tfvars` | Prod environment values |
